# Daily Tasks - HTML, CSS & JavaScript Course

This repository contains all the daily tasks and exercises for the **HTML, CSS, and JavaScript** course. It is organized into two main folders to keep your practice files structured and easy to access.

## Repository Structure
### 1. `webpages/`
This folder includes all the HTML pages and CSS stylesheets for the practice websites created during the course.

### 2. `js-tasks/`
This folder contains the JavaScript exercises and tasks assigned throughout the course. Each file corresponds to a specific task.
